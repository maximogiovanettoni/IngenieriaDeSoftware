## Consigna

Implementar el patrón Command para crear un sistema de control remoto que puede ejecutar diferentes comandos en un televisor.

Se necesita poder:
- Encender y apagar el televisor
- Cambiar de canal
- Aumentar y disminuir el volumen
- Deshacer la última acción (undo)

Los tests del archivo `RemoteControlTest.java` deben pasar.
