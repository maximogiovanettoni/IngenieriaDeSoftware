## Consigna

Implementar el patrón Abstract Factory para crear familias de objetos relacionados (UI de Windows y Mac).

Se necesita poder:
- Crear botones y ventanas específicas para Windows o Mac
- Asegurar que los componentes de la misma familia se usen juntos
- Permitir agregar nuevas familias sin modificar código existente

Los tests del archivo `UIFactoryTest.java` deben pasar.
